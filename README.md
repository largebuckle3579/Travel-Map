# Travel Map

Interactive world photo map with country galleries and Supabase-backed uploads.

## Setup

1. Run `supabase-setup.sql` in your Supabase SQL editor.
2. Add your Supabase project URL and publishable key in `assets/supabase-config.js`.
3. Open `index.html` locally or deploy the folder to Vercel.

Uploads work best from a real web URL such as Vercel, not from `file://`.
